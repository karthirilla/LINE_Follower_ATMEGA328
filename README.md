## üöÄ Features
- Auto Calibration
- Advanced Path Planning
- 14 Sensor Photodioda
- Layar Display
- Pilih Varian Motor DC 24mm kecepatan 13,000 rpm
- Driver Motor Mosfet
- Dual Memory Eeprom External 256Kbit
